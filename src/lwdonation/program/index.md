
# Apa itu Program

Untuk dapat menerima donasi, anda perlu membuat program donasi terlebih dahulu.
fungsi dari program donasi adalah untuk memberikan detail terkait donasi yang ingin anda dibuat.

Mulai dari 
- Judul
- Thumbnail
- Kategori
- Latar Belakang
- Target Donasi
- Deadline Donasi
- Minimal Donasi ( Jumlah Minimal Donatur Memberikan Donasi )
- Data Pencairan ( Untuk Transparansi )

Data data ini diperlukan untuk memberikan informasi kepada donatur terkait kegiatan apa yang sedang
membutuhkan donasi. ini juga berguna untuk transparansi kepada donatur. nantinya pada detail halaman donasi
akan di tampilkan juga daftar pencairan dan juga daftar donatur yang telah berdonasi.
